# Problem Record (Troubleshooting)

**Severity**: Low

## The Problem

- La HDS generada para receta 62 no incluia frases H/P, aunque el carbonato si las trae en la fuente.
- El generador por defecto creaba HDS para todas las MP (incluida Urea), lo que no era el objetivo.

## The Solution / Workaround

- Se modifico `scripts/generar_hds.py` para sumar frases H/P de todas las MP en el flujo `--receta-cod`.
- Se agrego filtro por COD y modo receta para evitar generar HDS no solicitadas.
